Reads 8 quadrature encoders and communicates over I2C.

https://github.com/RCMgames/BSCD

* Reads 8 quadrature encoders
* 1.0x0.85 inches
* I2C (Qwiic Compatible)
* powered by an ATMega328P
* hand solderable surface mount components
